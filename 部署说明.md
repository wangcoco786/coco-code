# AI-PM Platform 内网部署说明

## 快速启动（3 步）

### 第 1 步：安装 Node.js
下载并安装：https://nodejs.org（选 LTS 版本）

### 第 2 步：复制项目到目标机器
将整个 `ai-pm-platform` 文件夹复制到目标机器（任意位置）

### 第 3 步：双击启动
双击 `启动服务.bat`，等待出现内网地址

---

## 访问地址

启动后终端会显示：
```
内网访问:  http://192.168.x.x:3000  ← 分享给团队
```

团队成员在浏览器输入这个地址即可访问。

---

## 配置文件

编辑 `.env` 文件修改配置：

```
JIRA_BASE_URL=https://jira.logisticsteam.com
JIRA_USERNAME=你的Jira用户名
JIRA_PASSWORD=你的Jira密码
WECOM_WEBHOOK_URL=企业微信Webhook地址
PORT=3000
```

## 常见问题

**Q: 团队成员无法访问？**
A: 确保目标机器的防火墙允许 3000 端口。
Windows 防火墙设置：控制面板 → Windows Defender 防火墙 → 高级设置 → 入站规则 → 新建规则 → 端口 3000

**Q: 想换端口？**
A: 修改 `.env` 中的 `PORT=3000` 为其他端口

**Q: 想开机自启？**
A: 将 `启动服务.bat` 的快捷方式放入：
`C:\Users\用户名\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup`

**Q: Jira 数据加载失败？**
A: 检查 `.env` 中的 Jira 地址和账号密码是否正确

## 更新应用

当代码有更新时：
1. 替换项目文件
2. 运行 `npm install`（如有新依赖）
3. 运行 `npm run build`
4. 重启 `启动服务.bat`
