7. Department Performance

Five-Dimension Model (SPACE + DORA)

Throughput 20% — Completed tasks × complexity factor
Efficiency 25% — Cycle Time + on-time delivery rate
Quality 25% — Rework rate + Bug introduction rate
Impact 15% — High-priority completion + blocking resolution
Collaboration 15% — Cross-team comments + task participation

Performance Grades

80-100 Excellent 60-79 Good 40-59 Average 0-39 Needs Improvement

Score Formula

Score = Throughput×0.20 + Efficiency×0.25 + Quality×0.25 + Impact×0.15 + Collaboration×0.15
