7. 部门绩效

五维度评估模型（SPACE + DORA）

吞吐量 20% — 完成任务数 × 复杂度因子
效率 25% — Cycle Time + 按时交付率
质量 25% — 返工率 + Bug 引入率
影响力 15% — 高优完成比 + 阻塞解决速度
协作 15% — 跨团队评论 + 任务参与度

绩效等级

80-100 Excellent 60-79 Good 40-59 Average 0-39 Needs Improvement

综合绩效公式

绩效分 = 吞吐量×0.20 + 效率×0.25 + 质量×0.25 + 影响力×0.15 + 协作×0.15
